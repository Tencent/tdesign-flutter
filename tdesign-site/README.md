## 使用组件

### yaml引入依赖

```yaml
  dependencies:
    tdesign_flutter: ^0.1.0
```


### 引入组件代码

```dart
import 'package:tdesign_flutter/tdesign_flutter.dart';
```

## 本地运行官网示例代码

- 进入 `site` 文件夹，安装`npm install` 安装依赖包 
- `cd ..` 回到 `tdesign-site` 目录，运行命令 `npm run site:dev` 

## 基础库版本

最低基础库版本`^0.1.0`
