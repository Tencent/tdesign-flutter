## API
### TDBackTop
#### 默认构造方法

| 参数 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| controller | ScrollController? | - | 页面滚动的控制器 |
| key |  | - |  |
| onClick | VoidCallback? | - | 按钮点击事件 |
| showText | bool | false | 是否展示文字 |
| style | TDBackTopStyle | TDBackTopStyle.circle | 样式，圆形和半圆 |
| theme | TDBackTopTheme | TDBackTopTheme.light | 主题 |
